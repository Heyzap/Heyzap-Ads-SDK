HZAchievement Class Reference

Inherits from NSObject
Declared in HZAchievement.h

Overview

Model class representing an achievement on Heyzap, HZAchievements are provided in the achievement-related methods of HeyzapSDK. You would only need to use this class if you wanted to create a custom UI for showing achievements.
