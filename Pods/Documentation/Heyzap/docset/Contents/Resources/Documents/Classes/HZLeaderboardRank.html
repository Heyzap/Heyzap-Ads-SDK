Overview

HZLeaderboardRank is a model object describing a rank on a Heyzap Leaderboard. You receive these objects from network requests to Heyzap. You shouldn’t need to ever interact with HZLeaderboardRanks, unless you wish to create a custom leaderboard UI different from Heyzap’s.
